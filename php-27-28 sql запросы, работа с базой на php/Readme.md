1. Проверить поддерживает ли php работу с БД (php -m) pdo_sqlite sqlite3
2. Экспортировать вашу структуру БД в sql-Файл
3. Создать скрипт initDB.php который будет читать этот sql-файл структуры БД и выполнять его.
4. Устновить менеджер базы данных, или sqlite manager или использовать phpstorm, подключить к созданной БД
5. Добавить вручную тестовые данные блога.
6. Создать скрипт index.php в котором через query извлечть данные из БД в виде массива.
    а) Все посты, id и title
    б) Один пост (со всеми данными, категория юзер)
    в) Все категории
    г) Все посты одной категории

7* Извлеките комменты одного поста (текст коммента и автор)
